#Kanap

## Installation :

`Node` et `npm` doivent être installés localement sur votre machine.\
Installez toutes les dépendances avec npm.

Ouvrez le répertoire `back` depuis le terminal, puis :

```terminal
npm install
```

Pour lancer le server :

```terminal
node server
```
